URL Shortner API

User Story: I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
User Story: If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
User Story: When I visit that shortened URL, it will redirect me to my original link.

How it works! Simple Enough

Form you request like this.

http://localhost:8080/short/www.example.com or http://localhost:8080/g/https://www.example.com

Sever will respond like this.

{"original_url":"www.example.com","short_url":"http://localhost:8080/g/{url_number}"}

Now if you visit the short URL you will be redirected to the original website.
